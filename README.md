# Social-Disconnection-Identification-in-EHR
These files include all code to facilitate a LLM and supervised learning-guide NLP pipeline to identify social disconnection in the MIMIC-III Dataset. 

Files should be run in order of:
1. Lexicon Development
2. Regular Expression Matching
3. Classifiers (roberta, LLAMA-3, traditional ML)
